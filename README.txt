# ツール・ライブラリの名前
ニュース管理ツール

## 簡単な説明

TOPページからトレンドのニュースやいいね数でランキングが見れたりする
マイページに行くと、自分の投稿記事やいいねをした記事が見れる。


## 使い方

1. トップページ：森千夏
　 ログインしていてもしていなくても見ることができる
　 会員ログインボタンがある
2. ログインページ：藤井みゆき
　 会員であればログイン、非会員は新規登録
3. マイページ（投稿部分）：小針一軌
　 自分の記事を登録する
4．マイページ（投稿記事とお気に入り記事の閲覧）：北島由美恵
　 自分の投稿記事とお気に入り記事の一覧が見れる
